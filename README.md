**LAST SEEN LIVE**

A psychological mystery experience told through recovered livestream archives, chat logs, emails, 
and fragmented memories left behind by a missing streamer known only as Jinxed.
What begins as a simple investigation slowly turns into something far more personal.

As players solve interpretation-based clues and unlock hidden chapters, the line between observer and participant begins to blur. 
The deeper you go, the more Jinxed seems to know you, your habits, your curiosity, and most importantly…

why you stayed.

ABOUT THE GAME
LAST SEEN LIVE is a browser-based interactive horror narrative created using:
- HTML
- CSS
- JavaScript

This is not a traditional horror game focused on jumpscares or monsters. Instead, it explores:
- loneliness
- attachment
- parasocial relationships
- internet obsession
- emotional dependency
- the discomfort of being noticed for too long

The experience is intentionally minimal, quiet, and text-driven to create a more personal and immersive atmosphere.

FEATURES
- Interpretation-based clue solving
- Psychological storytelling
- Hidden chapters and archive pages
- Interactive emails and chat logs
- Minimalist horror atmosphere
- Mobile responsive design
- Progress tracking using localStorage

HOW TO PLAY

1. Open the game in your browser
2. Investigate archives, chats, and recovered files
3. Solve clues by interpreting patterns and emotions
4. Unlock hidden chapters
5. Discover the truth behind Jinxed Streams
